Artifact Summary
CS330
The CS 330 artifact is a 3D scene application from Computational Graphics and Visualization that was enhanced to demonstrate algorithms and data structures through a more organized, data-driven scene graph design and 
improved rendering logic.

CS340
The CS 340 artifact is the Grazioso Salvare project, which includes a Python CRUD module and dashboard application used to demonstrate both software design and engineering improvements and database-focused enhancements.

Purpose
This ePortfolio demonstrates growth in algorithms and data structures, software design and engineering, and databases through selected artifacts that were reviewed, enhanced, and documented as part of the 
CS 499 capstone experience.
